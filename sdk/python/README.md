# krxon

KRX (Korea Exchange) Open API Python SDK.

This package is generated by `krxon` and published as `krxon` on PyPI.

## Install

```bash
pip install krxon
```

## Usage

```python
from krx import KrxClient

client = KrxClient()
data = client.get_krx_index_daily(basDd="20260313")
print(data[0])
```
